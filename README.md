# Ninety Nine Global E-Commerce Platform

A premium, mobile-responsive luxury e-commerce platform for perfumes and watches.

## Project Structure

This repository contains two main applications:

- `frontend/`: The customer-facing storefront.
- `admin/`: The administrative control panel for managing products.

## Key Features

- **Luxury Interface:** High-end aesthetics, cinematic video headers, 3D-like elements, and smooth animations.
- **Multilingual Support:** English and Arabic with automated Right-to-Left (RTL) layout support.
- **Dual Currency:** Displays prices in LKR (Sri Lankan Rupee) and QAR (Qatari Riyal).
- **Product Management:** Dynamic gallery with multiple images per product, crossfade animations.
- **WhatsApp Integration:** Direct order system routing based on regional branch selection (Sri Lanka/Qatar).
- **Backend:** Powered by Firebase (Firestore).

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Running the Frontend

To start the customer-facing storefront:

```bash
cd frontend
npm install
npm run dev
```

### Running the Admin Panel

To start the administrative dashboard:

```bash
cd admin
npm install
npm run dev
```
